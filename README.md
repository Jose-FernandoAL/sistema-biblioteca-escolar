# Sistema de Biblioteca Escolar

## Sobre o Projeto

Sistema desenvolvido em Python com interface gráfica utilizando Tkinter para auxiliar no gerenciamento de bibliotecas escolares.

O sistema permite cadastrar livros e alunos, registrar empréstimos e devoluções, além de armazenar os dados em arquivos JSON.

## Funcionalidades

* Cadastro de livros
* Cadastro de alunos
* Registro de empréstimos
* Registro de devoluções
* Armazenamento em JSON
* Interface gráfica com Tkinter
* IDs automáticos

## Tecnologias Utilizadas

* Python
* Tkinter
* JSON

## Objetivo

Este projeto foi desenvolvido para praticar conceitos de programação, interfaces gráficas e organização de dados, simulando uma aplicação real utilizada em bibliotecas escolares.

## Como Executar

1. Instale o Python.
2. Baixe o projeto.
3. Execute:

```bash
python main.py
```
